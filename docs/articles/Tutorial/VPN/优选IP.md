CloudFlare 优选 IP 和域名适用于通过 CloudFlare 搭建的节点，或使用 CloudFlare 解析域名代理的节点，能够有效提升网络连接的稳定性和速度。下面介绍几种获取 CloudFlare 优选 IP 的方法。

### 方法一：使用 CloudFlare 优选 IP 工具

推荐使用 [CloudflareSpeedTest 工具](https://github.com/XIU2/CloudflareSpeedTest/releases)，支持 Windows、MacOS、Linux 等多个平台。该工具使用便捷、操作简单，并且能够快速筛选出最优的 CloudFlare IP 地址。

### 方法二：使用 CloudFlare 优选 IP 在线网站

如果你希望在手机上方便地选择优选 IP，可以尝试以下在线网站，它们支持 IPv4 和 IPv6，并可通过 Android 和 iOS 设备进行 IP 选择。

- 网站1：**https://stock.hostmonit.com/CloudFlareYes**
- 网站2：**http://ip.flares.cloud/**
- 网站3：**https://monitor.gacjie.cn/page/cloudflare/ipv4.html**

### CloudFlare 优选域名工具

对于优选域名，可以参考 GitHub 上的 [PingInfoView 项目](https://github.com/jeffreycayao/pinginfoview)，该工具能够帮助你快速检测并筛选出延迟最低的域名，从而获得更优质的网络体验。
